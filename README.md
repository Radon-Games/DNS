## Self Hosting

### Cloning
```
git clone https://github.com/Radon-Games/DNS.git
cd DNS
```

### Installing dependencies
```
npm install
```

### Configuring

Modify the `.env` file with the appropriate settings.

### Starting

```
npm start
```
